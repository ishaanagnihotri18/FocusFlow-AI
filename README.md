# 🚀 FocusFlow AI

An AI-powered productivity and task management application built using **React**, **Node.js**, **Express**, and **Google Gemini AI**.

## 📖 Overview

FocusFlow AI helps users organize tasks, prioritize deadlines, track productivity, and generate personalized schedules using Google's Gemini AI.

## ✨ Features

- Task Management
- AI Coach
- Gemini AI Productivity Planner
- Analytics Dashboard
- Settings Page
- Responsive UI

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- React Router
- Recharts

### Backend
- Node.js
- Express.js

### AI
- Google Gemini API

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/ishaanagnihotri18/FocusFlow-AI.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```

## 🔑 Environment Variables

Create a `.env` file inside the `server` folder.

```env
GEMINI_API_KEY=YOUR_API_KEY
```
